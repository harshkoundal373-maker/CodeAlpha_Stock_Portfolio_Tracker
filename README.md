# Stock Portfolio Tracker

## Overview

The Stock Portfolio Tracker is a Python-based project developed as part of the CodeAlpha Python Programming Internship. This application helps users calculate their total investment by selecting a stock and entering the quantity of shares they own.

The project demonstrates the use of Python fundamentals such as dictionaries, conditional statements, user input handling, loops, and arithmetic operations.

---

## Features

* Display available stocks
* Accept stock name and quantity from the user
* Calculate total investment value
* Show portfolio summary
* Display investment level based on total investment
* Simple and user-friendly interface
